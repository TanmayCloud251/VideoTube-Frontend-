# 📺 VideoTube Frontend

The modern, responsive user interface for the VideoTube platform, built with Next.js and Tailwind CSS.

## 🚀 Live Demo
- **Live Frontend:** [https://video-tube-frontend-sepia.vercel.app](https://video-tube-frontend-sepia.vercel.app)
- **Live Backend API:** [https://videotube-xby8.onrender.com](https://videotube-xby8.onrender.com)

---

## ✨ Features
- **Modern UI:** Clean, dark-themed interface using Tailwind CSS.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Dynamic Video Player:** Seamless video playback experience.
- **Authentication:** Integrated signup/login flow with "Remember Me" support.
- **Interactions:** Real-time feedback for likes, comments, and subscriptions.
- **Profile Management:** Custom channel pages with avatar and cover image updates.

---

## 🛠 Tech Stack
- **Framework:** [Next.js 15+](https://nextjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **API Client:** [Axios](https://axios-http.com/)
- **Icons:** [Lucide React](https://lucide.dev/)

## ⚙️ Getting Started

### Prerequisites
- Node.js 18.x or later
- A running instance of the VideoTube Backend

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/TanmayCloud251/VideoTube-Frontend-.git
   ```
2. Navigate to the frontend directory:
   ```bash
   cd "VideoTube(FrontEnd)/frontend"
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Create a `.env` file in the root of the frontend directory:
   ```env
   NEXT_PUBLIC_API_URL=https://videotube-xby8.onrender.com
   ```

### Running Locally
```bash
npm run dev
```
The application will be available at [http://localhost:3000](http://localhost:3000).
